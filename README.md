# TriathlonTraining
Triathlon Training app to allow me to practice android development techniques and assist me in training for triathlons

# Docs
### Installing NPM
run `brew install npm`
### Installing Docsify
run `npm i docsify-cli -g`
### Opening Docsify
`cd AndroidStudioProjects/TriathlonTraining/` (or wherever your android project folder is located) <br />
run `docsify serve .docs` in android project folder <br />
open [docs-link](http://localhost:3000/) <br />
